# perplexy chat app

## Description
Simple chat application using nextjs, axios and PerplexyAPI

## Installation

```bash
git clone https://github.com/nknighta/perplexy-chatapp-next.git
```
```bash
cd perplexy-chatapp-next
```
```bash
npm install
```
```bash
npm run dev
```

## Attention 
Please obtain a perplexy API key in advance
